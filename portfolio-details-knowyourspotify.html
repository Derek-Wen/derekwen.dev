Know Your Spotify

An interactive web application that tests your knowledge of songs and artists from your favorite Spotify playlists. Enter a Spotify playlist link and guess the song titles and artist names based on short audio previews.

Developed a front-end using HTML5, CSS3, and JavaScript (ES6) for interactive gameplay and responsive design, enhanced with Bootstrap 4 for styling and layout optimization. Integrated Chart.js to visualize game results with dynamic and visually engaging charts. Built a back-end with Node.js and Express.js, enabling robust server-side functionality and seamless handling of game sessions. Used Axios to interact with the Spotify Web API, fetching playlist data and audio previews for a personalized user experience. Designed the application to support playlist integration, random song selection, and progress tracking, delivering an engaging and dynamic music trivia experience.
